# ⚽ FIFA World Cup Analytics Dashboard (1970–2022) | Power BI

## 📌 Project Overview

This project presents a comprehensive **Power BI Dashboard** analyzing **FIFA Men's World Cup matches (1970–2022)**. The dashboard transforms historical football data into meaningful business insights through interactive visualizations, advanced DAX calculations, and predictive analytics.

The objective of this project is to demonstrate practical skills in **Data Analytics, Business Intelligence, Data Visualization, and Sports Analytics** using Microsoft Power BI.

---

## 🎯 Project Objectives

* Analyze FIFA World Cup performance from 1970–2022.
* Compare team performance across tournaments.
* Evaluate offensive and defensive efficiency.
* Analyze home (host) vs non-host performance.
* Explore historical tournament trends.
* Perform knockout stage analysis.
* Build advanced statistical metrics using DAX.
* Forecast future performance using historical trends.
* Provide AI-inspired business insights through Smart Narrative.
